    A. Pengertian HTML

          Sebagai bahasa markup, HTML membutuhkan sebuah fungsi atau tanda untuk memberitahu Web Browser apa yang perlu atau tidak perlu dilakukan. Tanda tersebut disebut dengan "Tag". Tag memiliki ciri-ciri ditulis di dalam "<....>". Kebanyakan tag dalam HTML ditulis secara berpasamham, yaitu tag pembuka dan tag penutup yang bisanya memiliki tanda "/" di dalam "<....>". Seperti contoh di bawah ini.

    <html> ..... </html>

          Namun ada juga tag yang berdiri sendiri, atau tidak berpasangan ( Single tag ), seperti tag untuk membuat baris baru.

    <br/>.....

          Element ialah semua data atau isi yang berada di dalam sebuah tag.

          Attribute ialah sebuah informasi yang diberikan pada tag, sebagai catatan atau bisa juga pengaturan untuk tag tersebut, biasanya dituliskan pada tag pembuka ( untuk yang berpasangan ) atau pada awal tag ( untuk single tag ). Seperti contoh berikut.

    <p align="center"> .... </p>

    B. Mengenal Structur Dasar HTML

          Seperti halnya setiap bahasa pemrograman yang sudah-sudah, HTML juga memiliki struktur dasar penulis HTML, diawali dengan tag DOCTYPE untuk menunjukkan pada browser type document dari file, kemudian dimulai dan ditutup dengan tag html, di dalam tag itu dapat diisikan tag head,body,dan lain sebagainya. Namun secara umum struktur dasar dari bahasa pemrograman HTML seperti contoh dibawah ini.

    kode pembuatan

        

       
      <!-- ini code html dasar -->

      <!DOCTYPE html>
      <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <!-- ini css nya -->
        <link rel="stylesheet" href="style.css">
        <!-- ini akhir css -->
      <title>Document</title>
      </head>
        <body>
            <header>Isi header</header>
            <p>Paragraf isi</p>
            <footer>Isi footer</footer>
        </body>
        </html>

    E. Cara Membuat Daftar/list di HTML <tag li>

          Dalam bahasa HTML jita dapat membuat urutan data atau dibuat menjadi "list". Tipe list itu sendiri dibagi menjadi dua, yaitu Order List atau pegurutan menggunakan angka dan Unordered List atau pengurutan berdasarkan simbol. Cara pemakaian Ordered List mengunakan Tag "<ol>.....</ol>" sedangkan untuk Unordered List menggunakan "<ul>....</ul>". Setelah dideklarasikan tipe pengurutannya, selanjutnya adalah mengisi urutannya, dengan menggunakan tag "<li>..........</li>" sebagai tanda per data.

    kode pembuatan

        
      <!-- ini code html dasar -->

      <!DOCTYPE html>
      <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <!-- ini css nya -->
        <link rel="stylesheet" href="style.css">
        <!-- ini akhir css -->
      <title>Document</title>
      </head>
        <body>
            <header>Isi header</header>
            <h1> Ordered List</h1>
            <ol>
               <li>Isi Data</li>
            </ol>
            <h1>Unordered List</h1>
            <ul>
               <li>Isi Data>
            </ul>
            <footer>Isi footer</footer>
        </body>
        </html>
    Check your code

          Simbol atau angka yang digunakan untuk tanda pengurutan pun dapat diganti menggunakan atribut "Type".

    F. Cara Membuat link di HTML <tag a>

          Salah satu fungsi yang paling sering digunakan adalah hypertext atau text yang jika diklik dapat terhubung dengan file atau halaman lain. Cara membuatnya dengan menggunakan tag "<a>.......</a>". Namun ada atribut dasar yang harus ada di dalam tag ini, yaitu "href" atau link dirctory yang dituju, dapat berupa ID, ataupu alamat. ID adalah suatu atribut dari sebuah tag yang digunakan untuk memberi nama atau identitas tag tersebut, dapat kita isi sesuai keinginan. Sebelumnya mencoba buatlah coding di bawah di sebuah file baru, terpisah dari file tutorial sebelumnya. Kemudian isi nilai dari "href" dapat diisi nama file dari tutorial sebelumnya.

    kode pembuatan

      
    <!-- ini code html dasar -->

    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta http-equiv="X-UA-Compatible" content="IE=edge">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <!-- ini css nya -->
      <link rel="stylesheet" href="style.css">
      <!-- ini akhir css -->
    <title>Document</title>
    </head>
      <body>
          <header>Isi header</header>
          <h1> Ordered List</h1>
          <a href="(NamaFile)"> Hyperlink on Tab</a>
          <br>
          <a href="(NamaFile)" target="_blank"> Hyperlink on Tab </a>
          <footer>Isi footer</footer>
      </body>
      </html>
    Check your code

          Dari kode di atas dapat dilihat atribut "Target" di dalam tag "<a>......</a>". Untuk mengatur hypertext itu terbuka di jendela yang sama atau di jendela yang baru, untuk New Tab menggunakan "_blank".
